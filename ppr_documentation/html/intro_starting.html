Getting started

This page describes how to get started using PoProofRead.

Opening the podiff-file

First download the file you wish to work with, e.g. "example.podiff". There are now three different ways that you can open the file:

  • Start PoProofRead and use the Open file dialog (Ctrl-o)

  • Right click the "example.podiff" file in your file browser and find PoProofRead among the applications in the Open with-menu

  • Start PoProofRead from the terminal and supply the file as a argument e.g: poproofread-gtk example.podiff

The PoProofRead main window

The figure below shows the PoProofRead main windown. Moving from the top down it consist of:

The navigation bar

From left to right it contains controls for setting and jumping to a bookmark, activating the in-line commenting mode, navigating through the diff parts or jumping to a specific diff part.

The diff and comment fields

The top and bottom fields in the center are used for showing the diff and typing in the comment, respectively. In in-line mode the comment field will expand and use all of the space.

The status line

This bottom line shows the status of the proofreading.

Proofreading in PoProofRead

To do a proofreading with the program follow these steps:

  1. Open the podiff you wish to proofread e.g. example.podiff

  2. Move though the document with the navigation buttons or the keyboard shortcuts

  3. When you encounter a diff part that you would like to comment on then simply start typing your comment

  4. If it is a large diff part and you wish to make comments in the middle of the diff part (in-line comments) then activate the in-line mode (with the control or the keyboard shortcut)

  5. When you are done with the proofreading, save your work, and the formatted output will be available in the file example.podiff.ppr.out. This will contain only the diff parts you commented on and your comments.
